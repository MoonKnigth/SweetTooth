# Project Overview: SweetTooth (Product List with Cart)
โปรเจกต์นี้คือระบบร้านขนมหวานออนไลน์ แบ่งเป็น 2 ส่วนคือ Frontend และ Backend

## Tech Stack Rules
1. **Frontend:** ใช้ Next.js (App Router), React, TypeScript, และ Tailwind CSS
2. **Backend (Future):** จะใช้ Laravel 11 สร้างเป็น REST API และใช้ฐานข้อมูล PostgreSQL
3. **Package Manager:** บังคับใช้ `pnpm` เท่านั้น ห้ามใช้ npm หรือ yarn เด็ดขาด
4. **Data Models:** โครงสร้าง Database ให้ยึดตาม ER Diagram ที่ออกแบบไว้ และฝั่ง Frontend ให้ยึดตาม TypeScript Interfaces ที่กำหนดให้
